Hide-Icons

Hide-Icons is a simple Windows tray app that lets you toggle desktop icons on or off with a click, set custom hotkeys, and control startup settings. Lightweight and built for a clutter-free desktop.

Visit the Repository Back to Hub

Key Features

Instant Icon Toggle

Hide or show your desktop icons with one click from the tray.

Custom Hotkeys

Set your own shortcut to toggle icons lightning-fast.

Startup Control

Choose if it runs when Windows boots—your call.
